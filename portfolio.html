Visualize Data with a Choropleth Map

This is FCC's "Visualize Data with a Choropleth Map" project found here. This project was to make a choropleth map based on education rates across counties in the US. I had no idea what a choropleth map was before this project so that was a neat thing to learn. This was also a challenge because there were multiple JSON files to load and just going off the data sets would put the incorrect county data into the graph so that took a bit to figure out how to get the correct counties to show. The tooltip for this project shows the county name and the rate of bachelors degrees or higher in said county.

Visualize Data with a Heat Map

This is FCC's "Visualize Data with a Heat Map" project found here. This is a heat map showing some variance to a base temperature. I'm not fully sure what it actually means though. Basically, each month in the years represented in the data has a variance to the base temp and the graph is color coded to show which month was higher and which was lower. You can hover over each box and a tooltip will show the year, and the variance.
